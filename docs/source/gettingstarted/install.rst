.. _install-label:

SDK 安装
================

环境要求
----------------------

-  Python 3.6+

正式版
----------------------

.. code-block:: shell

    pip install snbpy

开发版
----------------------

.. code-block:: shell

    pip install -i https://test.pypi.org/simple/ snbpy

源码安装
----------------------

.. code-block:: shell

    git clone git@github.com:snowballsecurities/snbpy.git
    cd snbpy
    python setup.py install
